# SnakeGame
A simple browser-based Snake game built with JavaScript, HTML, and CSS.

## Features
- Classic Snake gameplay
- Smooth keyboard controls (arrow keys / WASD)
- Score tracking
- Simple, lightweight codebase (vanilla JavaScript)
- Easy to customize (speed, grid size, colors)

## Controls
Arrow keys or WASD to move the snake
Space or P to pause/resume (edit if different)
